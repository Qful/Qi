## [enLKB简介](https://github.com/Qful/enLKB)

激光投影键盘，通过光学设计完成键盘投影，enLKB作为激光键盘的核心子系统，用于产生投影光路和接收并解析按键事件，工程将继承多个控制子系统。

本工程作为其他项目集成的一个组件，例如键盘和集成电脑设备等，同时也可以作为独立的工程实体进行开发。
